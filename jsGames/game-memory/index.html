Memory Game

Welcome to Memory Match, a fun and challenging game where you test your memory and speed. The game is based on matching pairs of cards with the same image.

To play the game, you need to flip over two cards at a time. If the cards match, you score 100 points and the cards stay face up. If the cards do not match, they flip back over and your error count increases by 1. You have 120 seconds to find all the matching pairs. But be careful, if you make more than 20 errors, you lose the game.

Can you beat the clock and match all the cards? Good luck and have fun!
